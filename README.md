# Netflix-Inspired-Frontend
Interface de streaming inspirada na Netflix, desenvolvida durante a Imersão Front-End da Alura para prática de HTML, CSS e JavaScript.

Este projeto foi desenvolvido durante a Imersão Front-End com Inteligência Artificial da Alura, com o objetivo de colocar em prática conceitos de HTML, CSS e JavaScript na criação de uma interface moderna e responsiva.

A proposta consiste em recriar uma tela inspirada em plataformas de streaming, como a Netflix, focando na experiência do usuário, organização de conteúdo e design visual atrativo.

🚀 Funcionalidades
Layout inspirado em plataformas de streaming
Interface responsiva para diferentes dispositivos
Organização de conteúdos em categorias
Estilização moderna com foco em UI/UX
🛠️ Tecnologias utilizadas
HTML5
CSS3
JavaScript

🎯 Objetivo
Praticar habilidades de desenvolvimento front-end, além de explorar criatividade no design e aplicação de conceitos modernos de interface.
